প্রজেক্টের নামঃ DigiTolls

Digitools একটি simple এবং user-friendly ওয়েবসাইট যেখানে বিভিন্ন দরকারি digital tools এক জায়গায় পাওয়া যায়। এই প্রজেক্টটা মূলত শেখার উদ্দেশ্যে তৈরি করা হয়েছে, যাতে React ব্যবহার করে কিভাবে একটি সুন্দর ও responsive website বানানো যায় সেটা প্র্যাকটিস করা যায়।

এই ওয়েবসাইটে বিভিন্ন ধরনের tools card আকারে দেখানো হয়েছে, যেখানে প্রতিটি tool-এর নাম, description, price এবং কিছু feature উল্লেখ করা আছে। ডিজাইনটা clean রাখার চেষ্টা করা হয়েছে যাতে user সহজে বুঝতে পারে এবং ব্যবহার করতে পারে।

পুরো project-এ modern frontend technology ব্যবহার করা হয়েছে, যেমন React, Tailwind CSS এবং কিছু UI component। Website-টা mobile এবং desktop দুই জায়গাতেই ভালোভাবে কাজ করার জন্য responsive করে তৈরি করা হয়েছে।

সব মিলিয়ে, Digitools একটি beginner-friendly project, যা real-world website বানানোর basic ধারণা দেয় এবং frontend development শেখার জন্য খুবই উপকারী।




এই প্রজেক্ট এ যা যা করা হয়েছেঃ 
১। React দিয়ে UI বানানো হয়েছে
২। Tailwind CSS দিয়ে design করা হয়েছে
৩। DaisyUI দিয়ে ready components ব্যবহার করা হয়েছে
৪। JSON data দিয়ে tools show করা হয়েছে
৫। Netlify দিয়ে host করা হয়েছে


এই প্রজেক্টের ৩ টি প্রধান বৈশিষ্ঠ রয়েছেঃ
১। একই জায়গায় বিভিন্ন দরকারি digital tools (AI tools, utilities, services) card আকারে দেখানো হয়েছে, যাতে user সহজে সব একসাথে পায়।
২। ওয়েবসাইটটি mobile, tablet এবং desktop—সব ডিভাইসে সুন্দরভাবে কাজ করে। UI automatically screen size অনুযায়ী adjust হয়।
৩। Tailwind CSS এবং DaisyUI ব্যবহার করে clean, modern এবং user-friendly design তৈরি করা হয়েছে, যাতে ব্যবহার করা সহজ হয়।








































# React + Vite

This template provides a minimal setup to get React working in Vite with HMR and some ESLint rules.

Currently, two official plugins are available:

- [@vitejs/plugin-react](https://github.com/vitejs/vite-plugin-react/blob/main/packages/plugin-react) uses [Oxc](https://oxc.rs)
- [@vitejs/plugin-react-swc](https://github.com/vitejs/vite-plugin-react/blob/main/packages/plugin-react-swc) uses [SWC](https://swc.rs/)

## React Compiler

The React Compiler is not enabled on this template because of its impact on dev & build performances. To add it, see [this documentation](https://react.dev/learn/react-compiler/installation).

## Expanding the ESLint configuration

If you are developing a production application, we recommend using TypeScript with type-aware lint rules enabled. Check out the [TS template](https://github.com/vitejs/vite/tree/main/packages/create-vite/template-react-ts) for information on how to integrate TypeScript and [`typescript-eslint`](https://typescript-eslint.io) in your project.
